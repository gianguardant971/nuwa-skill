## 📝 How to Use It

### 1. Choose a source

Pick one person, one role, or one topic at a time. This keeps the model clean.

### 2. Add real examples

Use text that shows how the person thinks in real situations. Short, clear examples work best.

### 3. Mark key habits

Look for:

- How they start
- What they ask first
- What they ignore
- How they decide
- How they explain the result

### 4. Review the model

Check whether the model sounds like the source. If it feels off, add more examples.

### 5. Refine over time

As you add more material, the model should get closer to the real pattern.

## 💡 Best Input Tips

Use source material that shows real thinking, not polished marketing text.

Good input:

- Direct answers
- Unscripted notes
- Raw interview transcripts
- Internal memos
- Voice-to-text drafts

Less useful input:

- Press releases
- Slogans
- Heavy editing
- Copy written by many people

## ❓ Troubleshooting

### The app does not open

- Check that the file finished downloading
- Run the app again as the same user
- Right-click the file and choose **Run as administrator**
- Make sure Windows did not block the file

### Windows SmartScreen appears

- Choose **More info**
- Then choose **Run anyway** if you trust the release

### The file is in a ZIP folder

- Extract the ZIP first
- Do not open the app from inside the compressed folder

### The app feels slow

- Close extra apps
- Restart your PC
- Use fewer large files at once
- Check that you have enough free memory and disk space
